# F103V2_encoder_motor_board
This is a 42 stepper motor control project using the F103C8T6 / F103CBT6.

1. Supports both CAN and RS485 communication.
2. Supports 5V / 12V endstop inputs.
3. Four-channel PWM output using SH1.0 4P connectors.
4. Supports data-command control only.
5. PID control with an MT6816 16-bit magnetic encoder.
6. The driver can be used with 42 / 57 stepper motors.
7. Uses the ME3116, with a maximum input voltage of 36V.
8. An OLED module can be added as an optional expansion.
9. Uses the TB67H450FNG motor driver, supporting up to 2.5A.
